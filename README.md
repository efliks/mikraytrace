# Mrtp

A minimalistic raytracing tool. 

Textures are from: https://opengameart.org/content/100-seamless-textures

Mrtp depends on the following libraries and tools (Debian packages):
 * libpng-dev
 * libpng++-dev
 * libeigen3-dev

Mrtp also depends on (submodules): 
 * cpptoml
